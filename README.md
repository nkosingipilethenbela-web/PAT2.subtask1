# Morse Code

## What is Morse Code?
Morse code is a system that uses dots (.) and dashes (-) to represent letters and numbers. It can be sent as sound, light, or electrical signals.

## History
Morse code was invented in the 1830s by Samuel Morse and Alfred Vail for use with the electric telegraph. It allowed people to send messages over long distances quickly for the first time.

## How It Works
Each letter has a unique pattern of dots and dashes:
- **Dot (.)** – short signal
- **Dash (-)** – long signal
- Letters are separated by a space
- Words are separated by three spaces
